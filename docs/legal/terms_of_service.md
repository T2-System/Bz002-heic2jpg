# NoTransfer Media Converter 利用規約

本利用規約(以下「本規約」といいます)は、[株式会社 T2 System](https://www.t2system.co.jp/) (以下「当方」といいます)が提供するアプリケーション「NoTransfer Media Converter」(開発上の名称を「HEIC2JPG」といい、以下あわせて「本アプリ」といいます)の利用条件を定めるものです。本アプリをご利用いただくユーザーの皆さま(以下「ユーザー」といいます)は、本規約に同意のうえ本アプリをご利用ください。

## 第1条(適用)

1. 本規約は、ユーザーと当方との間における本アプリの利用に関する一切の関係に適用されます。
2. ユーザーは、本アプリへのアクセス、インストール(PWA としての追加を含みます)または利用を開始した時点で、本規約に同意したものとみなされます。

## 第2条(定義)

本規約において使用する用語の意味は、次の各号に定めるとおりとします。

1. 「本アプリ」とは、当方が提供する NoTransfer Media Converter(開発上の名称 HEIC2JPG)という名称のアプリケーション、およびこれに付随する一切の機能・サービスをいいます。
2. 「ユーザー」とは、本アプリにアクセス、インストール、または利用する個人をいいます。
3. 「コンテンツ」とは、ユーザーが本アプリを通じて入力する変換対象のファイル(画像、動画、音声等)、本アプリが生成する変換結果、その他本アプリを通じて表示・処理される一切の情報をいいます。
4. 「外部サービス」とは、本アプリが機能提供のために連携または利用する、当方以外の第三者が提供するサービス(コンテンツ配信ネットワーク(CDN)、ライブラリ、SDK 等を含みます)をいいます。

## 第3条(本アプリの提供内容)

本アプリは、ユーザーの端末(ブラウザ)内で完結するメディア形式の変換を目的として、主として次の機能を提供します。

1. 画像形式の変換機能(HEIC 形式から JPG 形式への変換。EXIF 情報の保持および画質の調整を含みます)
2. 動画形式の変換機能(各種動画形式から MP4 形式への変換。リマックスまたは再エンコードを含みます)
3. 動画からの音声抽出機能(動画から MP3 形式の音声を抽出する機能)
4. 音声形式の変換機能(各種音声形式から MP3 形式への変換)
5. 複数ファイルの一括処理機能および変換結果の一括ダウンロード(Zip)機能
6. 表示言語、テーマ、出力品質、EXIF 情報の保持、変換方式等に関する設定機能
7. 前各号に付随する表示、補助および利便機能

1. 本アプリの利用にあたっては、ユーザーの責任と費用において、必要な通信機器、ソフトウェア、通信回線、対応ブラウザ等を準備するものとします。
2. 本アプリは、最新の Web 標準機能(SharedArrayBuffer、クロスオリジン分離(COEP/COOP)等)を前提とするため、対応するブラウザ環境でのみ正常に動作します。推奨環境その他の利用環境に関する情報は、本アプリまたは当方が適当と判断する方法により案内します。
3. 本アプリは、機能提供のために、ユーザーが選択したファイルへのアクセス、ドラッグ&ドロップによるファイルの受け取り、および変換結果のダウンロードに必要な権限を要求することがあります。本アプリは、カメラ、マイク、位置情報その他これらに類する権限を要求しません。
4. 本アプリで一度に処理できるファイルの数およびファイルサイズには上限が設けられており、当該上限を超える利用はできない場合があります。
5. 権限の不付与、通信環境の不備、対応外のブラウザまたはその設定その他利用環境の事情により、本アプリの全部または一部が利用できない場合があります。

## 第5条(外部サービスの利用)

1. 本アプリは、変換機能の提供に必要なソフトウェアモジュール(WebAssembly ライブラリ等)を取得するため、初回利用時その他必要な場合に、以下の外部サービスと通信を行うことがあります。
   - jsDelivr(@ffmpeg/ffmpeg を含む FFmpeg 関連モジュールの配信)
   - unpkg(@ffmpeg/util の配信)
   - その他、本アプリの動作に必要なライブラリの配信元(libheif 等。本アプリに同梱して提供する場合を含みます)

## 第7条(データの取扱い)

1. 本アプリに入力された変換対象のファイルおよびその変換結果は、原則としてユーザーの端末(ブラウザ)内でのみ処理され、当方のサーバーに送信されません。
2. 本アプリは、ユーザーの利便性のために、ブラウザのローカルストレージ(localStorage)等に次の情報を保存することがあります。
   - 表示言語に関する設定
   - テーマ(ライト、ダーク、自動等)に関する設定
3. 本アプリのダウンロード機能を利用した場合、変換結果はユーザーの端末に保存されます。当該保存先における取扱いは、ユーザーの端末、ブラウザまたは OS の仕様に従うものとします。
4. 本アプリは、オフラインでの利用その他の利便性のために、ブラウザのキャッシュおよび Service Worker にアプリの構成ファイルを保存することがあります。これらは、ブラウザのキャッシュ削除、本アプリ(PWA)のアンインストール、その他ユーザーの操作またはブラウザの仕様により消去されることがあります。
5. 第2項に基づき保存された情報は、ブラウザの設定、ブラウザのデータ削除、その他ユーザーによる操作により消去されることがあります。ただし、ユーザーが第3項に基づき端末に保存したデータについては、この限りではありません。
6. 本アプリは、第5条に定めるとおり機能提供に必要な範囲で外部サービスと通信を行いますが、当方が設計上想定する限り、当該通信に変換対象のファイルそのものまたはユーザーを直接特定する情報は含まれません。
